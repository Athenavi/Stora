# Stora 差距分析（实际状态）

> 对比目标：Google Drive / Dropbox / 百度网盘 / 阿里云盘 / 腾讯微云 / OneDrive  
> 基于实际代码审计结果，仅列出未实现或明显弱于行业标准的差距。

---

## 一、协作与团队

| 差距 | Stora 现状 | 行业标准 | 优先级 |
|------|-----------|----------|--------|
| **团队空间/群组** | ✅ Teams API (CRUD + 成员管理 + 共享文件夹) | Google Drive 共享云端硬盘、Dropbox 团队文件夹 | **P0 ✅** |
| **多人协作编辑** | ❌ 仅单人在线文本编辑 | Google Docs / Office 365 实时协作 | **P1** |
| **文件评论/批注** | ✅ List/Add/Delete 评论 API | Google Drive / Dropbox 文件评论 | **P2 ✅** |
| **分享链接到期自动清理** | ✅ 后台 30 分钟定时清理过期链接 | 百度网盘/阿里云盘 | **P2 ✅** |

## 二、预览与媒体

| 差距 | Stora 现状 | 行业标准 | 优先级 |
|------|-----------|----------|--------|
| **Office 在线编辑** | ⚠️ WOPI 端点就绪，需部署 Collabora Online + 完善文档 | 全支持 | **P1** |
| **图片编辑（裁剪/旋转/标注）** | ❌ 无图片编辑功能 | 百度网盘图片编辑 | **P1** |
| **视频转码** | ⚠️ transcode_tasks 表存在，ffmpeg 未对接 | 百度/腾讯自动转码多分辨率 | **P1** |
| **AI 字幕** | ❌ 无语音识别 | 百度/迅雷 AI 字幕 | **P2** |
| **文件水印（预览叠加）** | ❌ 无水印 | 百度网盘 | **P3** |

## 三、分享与传输

| 差距 | Stora 现状 | 行业标准 | 优先级 |
|------|-----------|----------|--------|
| **文件收集（他人上传到指定目录）** | ✅ POST /share/{code}/upload（upload 权限链接） | 百度网盘/阿里云盘"文件收集" | **P2 ✅** |
| **离线下载 BT** | ⚠️ HTTP 下载已实现，BT 未支持 | 百度/迅雷 | **P2** |
| **分享转存（保存到我的网盘）** | ✅ POST /share/{code}/save 已实现 | 百度网盘/阿里云盘 | **P2 ✅** |
| **二维码分享** | ✅ GET /share/{code}/qrcode（SVG 二维码） | 百度网盘 | **P3 ✅** |

## 四、安全与管理

| 差距 | Stora 现状 | 行业标准 | 优先级 |
|------|-----------|----------|--------|
| **完善的 Admin Web UI** | ⚠️ Go 模板基础面板，非完整管理后台 | 完善的 admin 后台 | **P1** |
| **Webhook 事件通知** | ❌ 无 webhook 系统 | Dropbox / Google Drive API | **P2** |
| **异地登录检测** | ❌ 无异常登录告警 | 百度/Google | **P2** |
| **IP 白名单访问控制** | ❌ 无 IP 限制 | Dropbox Business | **P3** |
| **存储配额超限预警** | ⚠️ quota 表存在，通知未对接 | 全支持 | **P2** |
| **文件访问权限（只读/评论/编辑）** | ⚠️ permission 三级（read/download/edit），edit 未实现 | Google Drive | **P2** |

## 五、系统运维

| 差距 | Stora 现状 | 行业标准 | 优先级 |
|------|-----------|----------|--------|
| **第三方登录（OAuth）** | ⚠️ GitHub/Google 端点存在，需配置完善 | 全支持 | **P2** |
| **批量用户导入/导出** | ❌ 无批量管理 | 全支持 | **P3** |
| **国际化和多语言 (i18n)** | ❌ 仅中文 | 全支持 | **P3** |
| **桌面同步客户端** | ❌ 无同步客户端 | 全支持 | **暂缓** |
| **移动 App** | ❌ 无移动端 | 全支持 | **暂缓** |

---

## 六、差距优先级汇总

| 优先级 | 差距项 | 估算 |
|--------|--------|------|
| **P0（当前）** | 团队空间/群组 | 12h |
| **P1** | Office 在线编辑、图片编辑、视频转码、Admin UI 完善 | 各 4-8h |
| **P2** | 协作编辑、文件收集、转存、评论、Webhook、异地登录、转码对接、AI字幕、分享编辑增强 | 各 2-6h |
| **P3** | 水印、IP白名单、批量导入导出、i18n、二维码、配额预警、桌面客户端、移动端 | 暂缓 |

## 七、实施路线

```
Phase 1 (已完成):  分享增强（收集/转存/二维码/到期清理） + 团队空间
Phase 2 (已完成):  文件评论/批注 + 设备管理/登录历史
Phase 3 (待实施):  Office 集成 + 图片编辑 + 视频转码
Phase 4 (待实施):  AI字幕 + 异地登录 + Webhook + 配额预警
```
