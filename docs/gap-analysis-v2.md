# Stora 综合差距分析 v2

> 对比目标：Google Drive / Dropbox / 百度网盘 / 阿里云盘 / 腾讯微云 / OneDrive  
> 仅列出 Stora 尚未实现或明显弱于竞品的差距项，已实现功能不列入。

---

## 一、存储基础

| 差距项 | Stora | 行业标准 | 优先级 |
|--------|-------|----------|--------|
| 上传/下载限速 | ✅ TokenBucket | 百度/迅雷支持 | **已完成** |
| 离线下载（URL/BT） | ✅ POST /files/offline-download + 后台 goroutine | 百度/迅雷 | **已完成** |
| 文件收集（他人上传到指定目录） | ❌ | 百度/阿里云盘 | P2 |
| 文件水印（预览时叠加） | ❌ | 百度网盘 | P3 |
| 文件对比/差异查看 | ❌ | Google Drive | P3 |

## 二、预览与媒体

| 差距项 | Stora | 行业标准 | 优先级 |
|--------|-------|----------|--------|
| Office 文档在线编辑 | ⚠️ WOPI 接口就绪，需部署 Collabora | 全支持 | P2 |
| 图片编辑（裁剪/旋转/标注） | ❌ | 百度网盘 | P2 |
| 视频转码进度 | ⚠️ 任务表存在，ffmpeg 未对接 | 全支持 | P2 |
| AI 字幕生成 | ❌ | 百度/迅雷 | P2 |
| 文档全文预览（非纯文本） | ⚠️ Flyfish 集成，但部分格式缺失 | 全支持 | P3 |

## 三、分享与协作

| 差距项 | Stora | 行业标准 | 优先级 |
|--------|-------|----------|--------|
| 分享密码保护 | ✅ bcrypt + 前端弹窗 | 全支持 | **已完成** |
| 分享有效期 | ✅ expires_in_hours + 前端选择器 | 全支持 | **已完成** |
| 分享下载次数限制 | ✅ max_downloads | 百度/阿里 | **已完成** |
| 批量下载 ZIP | ✅ BatchDownload 流式 ZIP | 全支持 | **已完成** |
| 团队空间/群组 | ❌ | Google Drive/OneDrive | P2 |
| 分享链接编辑（创建后修改） | ✅ PUT /files/shares/{id} | 全支持 | **已完成** |
| 分享到指定用户（按邮箱） | ✅ file_shares + ShareWithUser + 路由注册 | Google Drive | **已完成** |
| 协作编辑（多人同时） | ❌ | Google Drive/Office365 | P2 |
| 审核日志（谁访问了分享） | ⚠️ audit_logs 表存在 | Dropbox Business | P3 |
| 分享链接过期自动清理 | ❌ | 百度网盘 | P2 |

## 四、安全与管理

| 差距项 | Stora | 行业标准 | 优先级 |
|--------|-------|----------|--------|
| 两步验证 2FA | ✅ TOTP | 全支持 | **已完成** |
| 登录历史/设备管理 | ⚠️ user_sessions 表存在 | Google/微软 | P2 |
| 维护通知/公告 | ✅ 新实现 | 百度/阿里 | **已完成** |
| 异地登录检测 | ❌ | 百度/Google | P3 |
| 文件访问权限（只读/编辑/评论） | ⚠️ permission 三级存在，edit 未实现 | Google Drive | P2 |
| 存储配额超限预警 | ⚠️ quota 表存在，通知未对接 | 全支持 | P3 |
| IP 白名单访问控制 | ❌ | Dropbox Business | P3 |

## 五、系统运维

| 差距项 | Stora | 行业标准 | 优先级 |
|--------|-------|----------|--------|
| 管理后台（Web） | ⚠️ Go 模板基础面板 | 完善的 admin UI | P2 |

| Webhook 事件通知 | ❌ | Dropbox/Google | P3 |

| 国际化和多语言 (i18n) | ❌ | 全支持 | P3 |
| 桌面同步客户端 | ❌ | 全支持 | **暂缓** |
| 移动 App | ❌ | 全支持 | **暂缓** |

## 七、实施建议

```
Phase 1 (已完成) : 传输体验 + 分享基础
Phase 2 (当前)   : 团队空间 → 图片编辑 → 分享增强 → Office 集成
Phase 3 (后续)   : 媒体增值（AI字幕/转码）→ 安全增强（设备管理/IP白名单）
Phase 4 (长期)   : 生态建设（Webhook/i18n/客户端）
```
