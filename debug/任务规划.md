## 剩余任务详细规划

### P1.2 — OAuth 登录

```
工作量: 大 (需要 GitHub/Google OAuth App)
步骤:
  1. src/api/v2/auth/oauth.py — OAuth 回调处理
  2. 前端 /login 增加 "使用 GitHub 登录" 按钮
  3. .env 增加 GITHUB_CLIENT_ID / GITHUB_CLIENT_SECRET
  4. OAuth 成功后创建本地 User + 返回 JWT
```

### P2.2 — WebDAV 支持

```
工作量: 大 (协议实现复杂)
步骤:
  1. 安装 wsgidav 或自实现 PROPFIND/GET/PUT/DELETE
  2. 映射文件系统 API 到 WebDAV 操作
  3. 认证: JWT → HTTP Basic Auth 转换
```

### P2.3 — 在线文档编辑

```
工作量: 大
步骤:
  1. 部署 Collabora Online / OnlyOffice Document Server
  2. 实现 WOPI 协议端点
  3. 前端 iframe 嵌入编辑器
```

### P2.4 — Meilisearch 全文搜索

```
工作量: 中
步骤:
  1. 安装 meilisearch Python SDK
  2. 文件内容提取 (textract / python-magic)
  3. 索引创建 + 搜索 API
```

### P2.5 — 文件加密存储

```
工作量: 大
步骤:
  1. AES-256-GCM 加密/解密工具函数
  2. 上传时自动加密 (file_service.py)
  3. 下载时自动解密
  4. 密钥管理 (每用户密钥 / 主密钥)
```
