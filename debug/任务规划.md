## 当前完成状态

### 功能清单

```
用户认证:  注册 / 登录(JWT) / OAuth(GitHub+Google) / 2FA / 密码修改
文件管理:  CRUD / 上传(单文件+分片+秒传) / 下载(令牌) / 搜索 / 预览
文件夹:   CRUD / 树形 / 移动 / 导航面包屑 / 新建
分享:     创建(密码+过期+限制) / 列表 / 撤销 / 公开接收页 / 暗色页面
回收站:   列表 / 恢复 / 批量恢复 / 永久删除 / 清空
文件版本: 创建 / 列表 / 回滚
离线下载: 创建任务 / 列表
WebDAV:  PROPFIND / GET / PUT / DELETE / MKCOL / MOVE (HTTP Basic Auth)
WOPI:    在线文档编辑协议 (CheckFileInfo/GetFile/PutFile)
存储配额: 跟踪 / 侧边栏展示 / 硬限检查
存储套餐: StoragePlan 模型 / CRUD API
文件水印: 图片(居中/平铺) / PDF 叠加
加密存储: AES-256-GCM / HMAC 密钥衍生 (可选开关)
输入校验: 文件名 / 路径穿越 / XSS / 文件类型签名
错误处理: 统一错误码 / StoraException / 全局异常中间件
审计日志: 查看 / 筛选 API
系统:     设置 / 通知 / RBAC 权限 / 审计日志
前端:     Qwik + Tailwind v4 + SVG 图标库 / 暗色模式 / 响应式 / 骨架屏 / 懒加载
存储驱动: 本地 + S3/MinIO/R2
```

### 项目规模

```
Python 后端: ~169 文件 | 14 v2 路由模块 | 33 ORM 模型 | 11 服务模块
Qwik 前端:  10 路由页面 | 6 UI 组件 | SVG 图标库 | Tailwind v4
数据库:     stora2606 — 34 表 (含 storage_plans)
```
