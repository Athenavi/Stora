## 当前完成状态

### 已实现（全部可用）

```
用户认证: 注册 / 登录(JWT) / OAuth (Google+GitHub) / 2FA
文件管理: CRUD / 上传(单文件+分片+秒传) / 下载(令牌) / 搜索 / 预览(图片/视频/音频/PDF/文本)
文件夹:  CRUD / 树形 / 移动 / 导航面包屑
分享:    创建(密码+过期+限制) / 列表 / 撤销 / 公开接收页
回收站:  列表 / 恢复 / 批量恢复 / 永久删除 / 清空
文件版本: 创建 / 列表 / 回滚
离线下载: 创建任务 / 列表
WebDAV: PROPFIND / GET / PUT / DELETE / MKCOL / MOVE
WOPI:   在线文档编辑协议 (Collabora/OnlyOffice)
存储配额: 跟踪 / 展示
系统:    设置 / 审计日志 / 通知 / RBAC 权限
前端:    Qwik / Tailwind v4 / 暗色模式 / 8 页面 / SVG 图标库
存储驱动: 本地存储 + S3/MinIO/R2
```
