# Stora — 重构规划蓝图

> **战略目标**: 将 FastBlog（博客/CMS）彻底改造为 Stora（高性能网盘应用）
> **技术栈**: 后端 Python 3.14+ · FastAPI · SQLAlchemy 2.0 | 前端 Qwik · Tailwind CSS v4
> **生成时间**: 2026-06-17 | **更新**: 2026-06-17 | **当前进度**: Phase 0-4 基础框架已完成，进入前端细化 + 后端补充阶段

## 五、API 现状一览

```
✅ 已实现的文件 API:
  GET    /api/v2/files                   文件列表（分页/搜索/筛选）
  GET    /api/v2/files/{id}              文件详情
  PATCH  /api/v2/files/{id}              重命名/收藏
  DELETE /api/v2/files/{id}              删除（软删除/永久删除）
  POST   /api/v2/files/folders           创建文件夹
  PATCH  /api/v2/files/folders/{id}      重命名文件夹
  DELETE /api/v2/files/folders/{id}      删除文件夹
  GET    /api/v2/files/folders/{id}/children  文件夹子项
  GET    /api/v2/files/folders/tree      文件夹树
  POST   /api/v2/files/move              批量移动
  POST   /api/v2/files/upload            单文件上传（秒传）
  POST   /api/v2/files/upload/init       分片上传初始化
  POST   /api/v2/files/upload/chunk      上传分片
  POST   /api/v2/files/upload/complete   合并分片
  POST   /api/v2/files/upload/check      秒传检测
  GET    /api/v2/files/download/{id}     文件下载
  GET    /api/v2/files/download/token/{id}  下载令牌
  GET    /api/v2/files/search            文件搜索
  GET    /api/v2/files/search/history    搜索历史
  POST   /api/v2/files/shares            创建分享链接
  GET    /api/v2/files/shares            我的分享列表
  DELETE /api/v2/files/shares/{id}       撤销分享
  GET    /api/v2/files/shares/access/{code}  访问分享

⬜ 待实现的 API:
  POST   /api/v2/files/trash/{id}/restore  从回收站恢复
  GET    /api/v2/files/preview/{id}       文件预览
  GET    /api/v2/files/versions/{id}      文件版本历史
  POST   /api/v2/files/versions/{id}     创建新版本
  POST   /api/v2/admin/storage/plans     存储套餐管理
```
