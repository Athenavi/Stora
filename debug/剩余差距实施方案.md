## 3. P2.2 — 私密空间 Vault ✅

### 架构

```
独立 Vault 系统（不共享 Folder/FileItem 表）
├── Vault 表 — vault 元数据
├── VaultItem 表 — 加密文件记录
├── storage/vault/<user_id>/<vault_id>/ — 加密文件存储
└── AES-256-GCM 加密 + PBKDF2 密钥派生
```

### 密钥层次

```
用户密码 → PBKDF2(100k) → 派生密钥 → AES-GCM 解密 → 主密钥
                                                        ├── 加密每文件内容
                                                        └── 加密每文件名
```

### API 端点

| 端点 | 说明 |
|------|------|
| `POST /api/v2/vaults` | 创建 Vault |
| `GET /api/v2/vaults` | 列出 Vault |
| `DELETE /api/v2/vaults/{id}` | 删除 Vault |
| `POST /api/v2/vaults/{id}/verify-password` | 验证密码 → 返回 token |
| `PUT /api/v2/vaults/{id}/password` | 修改密码（O(1)，只重新加密主密钥） |
| `GET /api/v2/vaults/{id}/items` | 列出文件（需 X-Vault-Token） |
| `POST /api/v2/vaults/{id}/items/upload` | 上传并加密文件 |
| `GET /api/v2/vaults/{id}/items/{iid}` | 下载并解密文件 |
| `DELETE /api/v2/vaults/{id}/items/{iid}` | 删除文件 |

### 安全

- 数据库泄露 → 文件名加密看不到
- 存储泄露 → AES-GCM 密文
- 暴力破解 → PBKDF2(100k) + 独立 salt
- 密码丢失 → 数据不可恢复（UI 已明确提示）
